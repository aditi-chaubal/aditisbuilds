Heartbeat Sensor using Arduino

Project Overview

This is a heart rate monitor that uses a photoplethysmography sensor. I wrote the source code in C++ and it runs on an Arduino Uno. Pin-out diagram and pseudocode are below, and the full source code is on Github (linked below)

Pseudo Code

Program start

Initialize serial communication at 9600 baud
Define pulse sensor input pin

Create pulse detection object

Start infinite loop
    Read analog signal from pulse sensor pin
    Pass signal into detectBeat function

    In detectBeat:
        If signal rises above threshold AND latch is false
            Record current time
            If first beat OR time since last beat > minimum beat interval
                Mark beat as detected
                If previous beat exists
                    Calculate time between beats
                    Compute BPM
                    If BPM is within valid physiological range
                        Output BPM to serial monitor
                Update last beat time
            Set latch to true

        If signal falls below threshold
            Reset latch

Repeat infinite loop
